개인석 입/퇴실
입실하기
1. 사용자가 로그인을 하면 자동으로 개인석을 예약할 수 있는 화면으로 이동합니다.
2. 현재 개인석을 이용중인 현황을 확인할 수 있으며, 이용불가 좌석은 선택이 불가하도록 구현했습니다.
3. 원하는 번호의 좌석을 선택하면 입실을 진행할 수 있는 팝업이 뜹니다.
4. 이용권이 존재한다면 사용중인 상품을 클릭하여 현재 남은 시간을 확인할 수 있고 사용 여부의 팝업을 띄워 입실을 진행합니다.
5. 만약, 사용중인 상품이 없다면 이용권을 구매하도록 팝업을 띄웁니다.
6. 입실 절차가 완료되었다면, 자신이 선택한 좌석의 색이 주황색으로 바뀝니다.
퇴실하기
1. 사용자가 자신이 이용중인 좌석을 선택하면 자리이동, 퇴실하기, 이용권 연장을 선택할 수 있는 팝업이 뜹니다.
2. 퇴실하기 버튼을 클릭하면 현재 사용자의 남은 이용권 시간이 안내되며, 이용중이었던 좌석의 색상이 변경됩니다.
3. 좌석배치도 왼쪽 상단의 잔여 좌석 수도 변경된 것을 확인할 수 있습니다.
RESPONSIBILITY

개인석 자리이동
개인석 자리이동하기
1. 사용자가 자신의 좌석을 보유하고 있는 상태에서 자신의 좌석을 선택하면 좌석이동을 진행할 수 있는 팝업이 뜹니다.
2. 좌석 이동 버튼을 클릭하면 좌석 배치도 상단에 '원하시는 좌석을 선택해주세요.'로 문구가 변경되며 원하는 좌석을 선택할 수 있습니다
3. 원하는 좌석을 선택할 경우, 좌석 이동 여부 팝업을 띄우며 확인을 누르면 좌석이동이 완료되어 좌석 배치도의 색상이 변경됩니다.
RESPONSIBILITY
담당 역할 상세 소개

스터디룸 예약
스터디룸 - 예약 현황 조회하기
1. 사용자가 왼쪽 스터디룸 버튼을 클릭하면 패널이 변경됩니다.
2. 스터디룸은 1번과 2번 스터디룸이 존재하며 화살표를 클릭하여 스터디룸을 변경할 수 있습니다.
3. Default는 현재 날짜로 맞춰지며, 현재 날짜 이후의 예약 현황도 조회가 가능합니다.
4. 현재 날짜를 기준으로 이전 날짜는 조회할 수 없도록 버튼에 비활성화를 적용합니다.
5. 현재 시간을 기준으로 이전 시간은 이용불가 색상으로 변경되며 실시간으로 시간이 지날 때마다 이용불가의 색상이 점점 채워집니다.
6. 다른 사용자가 미리 예약을 한 시간대는 노란색으로 채워지는 것을 볼 수 있습니다.
스터디룸 - 예약하기
1. 사용자가 원하는 스터디룸과 날짜, 이용 시간을 지정할 수 있도록 화살표 버튼이 존재합니다.
2. 원하는 설정을 마친 후 타임 테이블에서 원하는 시간대를 15분 단위로 선택할 수 있습니다.
3. 이때, 이용시간은 최소 1시간이기때문에 다른 사용자가 예약을 해놓은 시간을 기준으로 앞 3칸은 선택을 할 수 없습니다. (2시간 이용을 선택하면 앞 7칸을 비활성화)
4. 이외에 원하는 시간대를 클릭하면 주황색으로 변경되는 것을 볼 수 있으며, 결제하기 버튼이 활성화되는 것을 확인할 수 있습니다.
